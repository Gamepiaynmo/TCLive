# TCLive

Baidu Tieba Live Mod for Minecraft

## Input

- Use @[pic%d]@ to insert pictures
- Use @[emo%d]@ to insert emotions
- Use \n to start a new line

## Tutorial

[Link](https://tieba.baidu.com/p/5114045077)

## Note

- Java 1.8 required
- Try the new login method !

## Supported Versions

- Minecraft 1.7.10
- Minecraft 1.10.2
- Minecraft 1.11.2
